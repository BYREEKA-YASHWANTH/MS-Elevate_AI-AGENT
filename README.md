# AI Student Helpdesk Agent

## Description
The AI Student Helpdesk Agent is a conversational assistant designed to help students with common academic and campus-related queries. It provides guidance on topics such as time management, attendance concerns, exam preparation, and communication.

## Technologies Used
- Python
- AI Prompt Design
- Conversational AI

## Features
- Guidance for attendance and missed classes
- Exam preparation and stress management tips
- Time management and semester planning support
- Email writing and communication assistance
- General campus guidance

## Project Type
AI-based Student Support Assistant
